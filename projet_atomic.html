Atomic

Site d'hébergement vidéo | Média social

Atomic Video est un site web d'hébergement vidéo. Parmis ses fonctionnalités, nous pouvons compter :

  • Hébergement de vidéos et miniatures
  • Gestion de comptes utilisateurs
  • Commentaires et abonnements
  • Gestion des vues

Voir le PDF

Description

En fin de BTS, un projet PHP en binaire était requis pour valider le diplome. Ainsi, nous avons développé Atomic, une plateforme pour partager ses vidéos, les commenter, s'abonner et plus.

Fonctionnalités

A l'image de Youtube, vous pouvez créer un compte et le gérer depuis Atomic. Ce compte vous permet de poster des vidéos comportant un titre, une description, un genre... Il vous permet aussi de "liker" les vidéos, les commentaires, répondre à ces commentaires...

Structure

Le site est hébergé avec Apache2 sur une machine Ubuntu Server.

Il est relié à une base de données MySQL à l'aide du langage PHP.
